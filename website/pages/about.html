Início

Missão

O My-co-Lab nasceu como um projeto pessoal para explorar o cultivo de cogumelos, a recolha de dados ambientais e a documentação técnica.

O objetivo não é apenas produzir cogumelos, mas compreender o processo.

Tecnologias

  • Raspberry Pi
  • Raspberry Pi Pico
  • Sensores ambientais
  • Fotografia e timelapse
  • Python
  • Documentação contínua

Estado do Projeto

  • Grow box em desenvolvimento
  • Sistema de sensores operacional
  • Website em construção
  • Primeira frutificação em preparação

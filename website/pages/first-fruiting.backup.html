O que é o My-co-Lab?

O My-co-Lab é a parte prática e experimental deste site: uma grow box inteligente usada para acompanhar o crescimento dos cogumelos.

Primeira frutificação

O objetivo é acompanhar o ciclo completo de crescimento dos cogumelos e documentar ambiente, imagens, timelapse e observações.

Setup utilizado

  • Raspberry Pi 4
  • Raspberry Pi Pico
  • DHT22
  • Câmara
  • Humidificador
  • Timelapse
  • Registo de dados ambientais

Sobre o Projeto

Um site para organizar e partilhar conhecimento sobre cogumelos.

Início Micologia My-co-Lab

Missão

O objetivo deste projeto é criar uma plataforma educativa sobre cogumelos, organizando conhecimento de forma simples, prática e progressiva.

O foco principal é aprender e explicar: o que são fungos, como vivem, como se identificam, como se cultivam e qual o seu papel na natureza.

Fontes de conhecimento

  • Livros de micologia
  • Guias de identificação
  • Livros de cultivo
  • Observações pessoais
  • Experiências práticas
  • Base de dados própria

O papel do My-co-Lab

O My-co-Lab é a parte experimental do projeto: uma grow box inteligente usada para observar cultivos reais com sensores, fotografias e timelapse.

Ele não é o tema principal do site, mas é aquilo que torna este projeto mais prático, original e diferente.

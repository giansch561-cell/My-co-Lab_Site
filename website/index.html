O que é o My-co-Lab?

O My-co-Lab é um projeto pessoal que junta cultivo de cogumelos, eletrónica, Raspberry Pi, sensores, programação e documentação.

O objetivo é criar uma grow box inteligente capaz de acompanhar o crescimento dos cogumelos com dados ambientais, fotografias e timelapse.
